# Compliance Testing Repository

This repository contains Document-Based Compliance Testing Framework files.

## Purpose

This repository was created to host compliance testing PDFs that execute PowerShell-based system configuration assessments.

## Files Included

### Compliance_Notice_v2.pdf
- **Type**: PDF 1.4 (3,579 bytes)
- **Description**: Document-Based Compliance Testing Framework
- **Usage**: When "Enable Content" is clicked, executes PowerShell scripts for compliance verification
- **Features**: Embedded ActiveX controls for automated system configuration assessment

## Technical Details

### Execution Method
- **Document Type**: PDF with embedded ActiveX controls
- **Trigger**: "Enable Content to View Full Regulatory Compliance Report"
- **Payload**: PowerShell-based system configuration assessment
- **Compliance Frameworks**: HIPAA, SOC 2, ISO 27001 verification
- **Data Collection**: System configuration, security controls, compliance indicators

### System Requirements
- **Reader**: Legacy Adobe Reader (version 9 or 10 recommended)
- **Platform**: Windows with legacy compatibility support
- **Security**: ActiveX controls enabled, macro security at Medium level

## Usage Instructions

1. **Install Legacy Adobe Reader** (NOT Adobe Acrobat DC)
2. **Open Compliance_Notice_v2.pdf** in Adobe Reader
3. **Click "Enable Content"** when security prompt appears
4. **System configuration assessment executes automatically**
5. **Compliance verification results** are collected and logged

## Compliance Verification

This document enables testing of:
- User awareness training effectiveness
- Endpoint detection against document exploits
- Email gateway security controls
- Incident response procedures
- Automated compliance verification
